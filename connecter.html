SAE 1.02 : Connecter

Conception et mise en œuvre d'une infrastructure réseau virtuelle

Fiche Projet

  • Logiciels : Draw.io, Cisco Packet Tracer
  • Systèmes : Linux Ubuntu (Serveur Apache)
  • Gestion : Diagramme de Gantt
Services déployés :
HTTP (Apache) DNS DHCP

De la simulation à la virtualisation

Ce projet marque une étape clé : le passage de la théorie à la pratique. L'objectif est de concevoir une architecture réseau complète, de la planifier rigoureusement, puis de la rendre fonctionnelle.

1. Conception & Planification

  • Réalisation d'un diagramme de Gantt.
  • Modélisation de l'infrastructure via Draw.io.
  • Recherche documentaire sur les protocoles.

2. Simulation & Test

  • Configuration logicielle des routeurs.
  • Capture et analyse de trames.
  • Rédaction d'un compte-rendu technique.

Mise en œuvre réelle (Virtualisation)

Déploiement final sur Ubuntu :

  • Serveur Web : Installation d'Apache2.
  • Services IP : DNS et DHCP via Dnsmasq.
  • Tests : Preuve de fonctionnement via Wireshark.
