Transformer la donnée brute en information visuelle

Ce projet consiste à réaliser un script capable d'automatiser le traitement de données ouvertes issues de data.gouv.fr.

Étapes du pipeline de données

  • Extraction : Récupération automatisée de fichiers CSV.
  • Agrégation : Calcul de moyennes, maximums et filtrages via Python.
  • Transformation : Conversion des résultats au format JSON.
  • Visualisation : Transmission à l'API QuickChart pour rendu graphique.

Méthodologie et Collaboration

Le projet repose sur une utilisation rigoureuse de GitLab pour le stockage des scripts et de la documentation (Markdown).

  • Planification des tâches et répartition équitable du travail.
  • Utilisation de Git pour le suivi des versions (commit/push individuels).
  • Rédaction d'un fichier README.md pour la documentation utilisateur.
Bilan : Ce projet valide ma capacité à coder une solution complète de traitement de données en équipe.
