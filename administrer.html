SAE 1.04 : Administrer

Hébergement, gestion de contenu et déploiement d'un service web

Gestion de Service

  • Compétence : Administrer
  • Plateforme : GitHub Pages
  • Outil de versioning : Git
  • Statut : Déploiement opérationnel
Compétences mises en œuvre :
Administration Web Gestion de dépôt Git Identité Numérique

Administrer son image et ses services sur Internet

Dans le cadre de la compétence Administrer, ce projet m'a permis de mettre en place une infrastructure complète pour diffuser mon portfolio professionnel. Il ne s'agissait pas seulement de coder, mais de gérer l'accessibilité et la pérennité d'un service en ligne.

Déploiement et Maintenance

L'administration de ce site repose sur des outils professionnels utilisés dans le monde de l'informatique :

  • Configuration de l'hébergement : Utilisation de GitHub Pages pour rendre le service accessible via une URL publique.
  • Gestion de version (Git) : Mise en place d'un workflow avec les commandes git commit et git push pour administrer les mises à jour de manière sécurisée.
  • Organisation des fichiers : Gestion rigoureuse des chemins relatifs et des ressources pour assurer la stabilité du site.

Objectifs de la SAE 1.04

Ce projet encadré par David Mercier et Serigne Mamadou Diène visait à maîtriser l'ensemble de la chaîne de production d'un site web professionnel : du choix du template Bootstrap au respect des normes W3C, jusqu'à la mise en ligne finale.

Bilan : Ce travail prouve ma capacité à administrer un service web simple et à utiliser des outils de versioning essentiels pour tout technicien Réseaux & Télécoms.
