SAE 1.04 : Administrer

Hébergement, gestion de contenu et déploiement d'un service web

Gestion de Service

  • Compétence : Administrer
  • Plateforme : GitHub Pages
  • Outil de versioning : Git
  • Statut : Opérationnel
Compétences clés :
Administration Web Versioning Git Responsive Design

Administrer son image et ses services sur Internet

Dans le cadre de cette SAE, j'ai mis en place une infrastructure complète pour diffuser mon portfolio. Administrer, ici, signifie gérer la mise en ligne, le stockage et l'accessibilité d'un service web.

Déploiement et Maintenance

Le pilotage de ce projet repose sur des outils standards du monde professionnel :

  • GitHub Pages : Configuration du serveur d'hébergement pour rendre le site accessible via une URL publique.
  • Gestion de version (Git) : Utilisation des commandes commit et push pour administrer les mises à jour en toute sécurité.
  • Rigueur technique : Respect des normes W3C et gestion des chemins relatifs pour assurer la stabilité du site.

Objectifs pédagogiques

Encadré par David Mercier et Serigne Mamadou Diène, ce travail m'a permis de maîtriser la chaîne de production d'un site web, de l'adaptation du code Bootstrap au déploiement final.

Bilan : Ce projet démontre ma capacité à utiliser des outils de versioning essentiels et à administrer un service web fonctionnel.
